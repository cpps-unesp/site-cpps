<Notice type="tip">
Esta parada da Trilha de Dados trata da análise de dados utilizando a linguagem de programação Python.

O estudante terá contato com bibliotecas como `pandas`, `numpy`, `matplotlib` e `plotly`.
</Notice>

## Pré-requisitos

- [Treinamento de Preparação de Ambiente de Trabalho](/pt/wiki/projetos/ensino/tfdt/trilha/paradas/ambiente/01-intro)
- [Fundamentos de programação com Python](/pt/wiki/projetos/ensino/tfdt/trilha/paradas/fundamentos-python/01-intro)
- [Coleta de dados com Python](/pt/wiki/projetos/ensino/tfdt/trilha/paradas/coleta-dados/intro) — fornece o dataset usado nos exemplos
