<Notice type="tip">
Esta parada da Trilha de Dados trata da análise de dados utilizando a linguagem de programação Python.

O estudante terá contato com bibliotecas como `pandas`, `numpy`, `matplotlib` e `plotly`.
</Notice>

## Pré-requisitos

- [Treinamento de Preparação de Ambiente de Trabalho](/atividades/projetos/ensino/trilha-dados/ambiente/intro)
- [Fundamentos de programação com Python](/atividades/projetos/ensino/trilha-dados/fundamentos-python/01-intro)
- [Coleta de dados com Python](/atividades/projetos/ensino/trilha-dados/coleta-dados/intro) — fornece o dataset usado nos exemplos

## Tópicos do treinamento

- [**01 — Conceitos básicos**](./01-00-conceitos-basicos): NumPy vs Pandas, instalação, Series e DataFrame.
- [**01-01 — Series e DataFrame**](./01-01-pandas-series-e-dataframe): criação manual e a partir de dicionários/listas.
- [**01-02 — Propriedades básicas**](./01-02-propriedades-basicas): `shape`, `dtypes`, `head`, `info`, `describe`, `value_counts`.
- [**01-03 — Consulta e modificação**](./01-03-consulta-e-modificacao): `loc`/`iloc`, filtros booleanos, `.str`, novas colunas.
- [**01-04 — Arquivos**](./01-04-trabalhando-com-arquivos): `read_csv`, `read_json`, `json_normalize`, exportação.
- [**02 — Gráficos**](./02-gerar-graficos): Matplotlib, Plotly Express, casos reais com notas do MRE.
- [**03 — Combinando DataFrames**](./03-combinando-dataframes): `concat`, `merge`, `join`, junções.
- [**04 — Tratamento**](./04-tratamento-de-dados): valores faltantes, conversão de tipos, duplicatas, `groupby`.
