<Notice type="tip">
Esta parada da Trilha de Dados trata da coleta de dados utilizando a linguagem de programação Python.

O estudante terá contato com bibliotecas como `requests`, `BeautifulSoup`, `Scrapy`, `Selenium` e `TinyDB`.
</Notice>

## Pré-requisitos

- [Treinamento de Preparação de Ambiente de Trabalho](/atividades/projetos/ensino/trilha-dados/ambiente/intro)
- [Fundamentos de programação com Python](/atividades/projetos/ensino/trilha-dados/fundamentos-python/01-intro)

## Tópicos do treinamento

- [**Tópico 01 — Conceitos iniciais**](./01-conceitos): fontes de pesquisa, estrutura de páginas web (HTML/CSS), inspeção de elementos, página estática vs. dinâmica.
- [**Tópico 02 — `requests` + `BeautifulSoup`**](./02-requests-bs4): requisições HTTP, parsing, `find` e `find_all`, atributos.
- [**Tópico 03 — Paginação e erros**](./03-paginacao-e-erros): percorrer páginas, `try/except`, retries, pausas entre requisições.
- [**Tópico 04 — Tratamento**](./04-tratamento): `strip()`, `split()`, conversão de tipos, datas.
- [**Tópico 05 — Armazenamento**](./05-armazenamento): JSON, `TinyDB`, deduplicação, arquivos binários.
- [**Tópico 06 — Selenium**](./06-paginas-dinamicas-selenium): páginas dinâmicas, headless, interação com formulários.
