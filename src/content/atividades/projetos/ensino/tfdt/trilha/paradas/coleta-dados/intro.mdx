<Notice type="tip">
Esta parada da Trilha de Dados trata da coleta de dados utilizando a linguagem de programação Python.

O estudante terá contato com bibliotecas como `requests`, `BeautifulSoup`, `Scrapy`, `Selenium` e `TinyDB`.
</Notice>

## Pré-requisitos

- [Treinamento de Preparação de Ambiente de Trabalho](/atividades/projetos/ensino/trilha-dados/ambiente/intro)
- [Fundamentos de programação com Python](/atividades/projetos/ensino/trilha-dados/fundamentos-python/01-intro)

## Tópicos do treinamento

- **Tópico 01 — Conceitos iniciais**: fontes de pesquisa, estrutura de páginas web (HTML/CSS), inspeção de elementos.
- **Tópico 02 — Extração de informações**: requisições HTTP com `requests`, parsing com `BeautifulSoup`, navegação por seletores.
- **Tópico 03 — Paginação e tratamento de erros**: percorrer múltiplas páginas, blocos `try/except` para lidar com elementos ausentes.
- **Tópico 04 — Tratamento de dados**: limpeza e normalização de strings com `strip()`, `split()`, conversão de tipos.
- **Tópico 05 — Armazenamento**: persistência em JSON e em banco de dados leve com `TinyDB`.
