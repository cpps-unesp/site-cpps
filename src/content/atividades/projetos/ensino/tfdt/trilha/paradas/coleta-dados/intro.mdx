<Notice type="tip">
Esta parada da Trilha de Dados trata da coleta de dados utilizando a linguagem de programação Python.

O estudante terá contato com bibliotecas como `requests`, `BeautifulSoup`, `Scrapy`, `Selenium` e `TinyDB`.
</Notice>

## Pré-requisitos

- [Treinamento de Preparação de Ambiente de Trabalho](/pt/wiki/projetos/ensino/tfdt/trilha/paradas/ambiente/01-intro)
- [Fundamentos de programação com Python](/pt/wiki/projetos/ensino/tfdt/trilha/paradas/fundamentos-python/01-intro)
