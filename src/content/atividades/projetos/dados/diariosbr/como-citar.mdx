Se você utilizou os dados ou a documentação do projeto **DiáriosBR** em sua pesquisa, artigo ou trabalho acadêmico, pedimos que cite o projeto. Isso é fundamental para dar crédito aos pesquisadores e garantir a continuidade do financiamento e suporte ao projeto.

Abaixo estão os modelos de citação nos formatos mais utilizados.

## Formato ABNT

> MARIANO, Marcelo Passini (Coord.). **DiáriosBR**. Laboratório de Relações Internacionais (LABRI-UNESP). Disponível em: &lt;https://labriunesp.org/docs/projetos/dados/diariosbr&gt;. Acesso em: [dia] [mês] [ano].

## Formato APA

> Mariano, M. P. (Coord.). (2021). _DiáriosBR_. Laboratório de Relações Internacionais (LABRI-UNESP). https://labriunesp.org/docs/projetos/dados/diariosbr

## BibTeX

Para usuários de LaTeX ou gerenciadores de referência (Zotero, Mendeley, EndNote), utilize o código abaixo:

```bibtex
@misc{diariosbr,
  author = {Mariano, Marcelo Passini},
  title = {DiáriosBR},
  year = {2021},
  publisher = {Laboratório de Relações Internacionais (LABRI-UNESP)},
  url = {https://labriunesp.org/docs/projetos/dados/diariosbr},
  note = {Acessado em: [Data de Acesso]}
}
```

Os dados disponibilizados pelo DiáriosBR são de uso público para fins de pesquisa acadêmica e não comercial. Ao utilizar os dados, o pesquisador concorda em citar a fonte conforme indicado acima.
